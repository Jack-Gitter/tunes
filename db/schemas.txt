DATABASE SCHEMAS

Users
    spotifyID (pk)
    username
    followers
    following

Posts
    postID (pk)
    spotifyID (fk)
    trackID
    postContent
    rating
    likes
    dislikes
    
Comments
    commentID (pk)
    parentCommentID (fk)
    postID (fk)
    spotifyID (fk)
    likes 
    dislikes

ACTUAL JSON: 

User {
    spotifyID: int
    followers: List<spotifyID>
    following: List<spotifyID>
    posts: List<postID>
    comments: List<commentID>
}

Post {
    postID: int
    userID: int
    trackID: int
    postContent: string
    rating: int
    likes: int
    dislikes: int
    comments: List<Comment>
}

Comment {
    commentID: int
    postID: int
    parentCommentID: int
    spotifyID: int
    childComments: List<Comment>
    likes: int
    dislikes: int
}
